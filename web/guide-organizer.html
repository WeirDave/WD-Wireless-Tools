WD Squirrel

Stash your Ekahau files — sort loose files into organized subfolders, create new project folders with the right structure.

Getting Started

Step 1
Pick a Folder
Click Browse to select a root folder containing site subfolders, or a single folder with loose files.
Step 2
Scan
The tool scans every subfolder, classifies files by extension, and shows a preview of proposed moves.
Step 3
Review
Check the preview, adjust destinations if needed using the per-file dropdowns.
Step 4
Organize
Click Organize to move files into their destination subfolders.

You can enter a folder path directly using backslash notation (e.g. C:\Sites\ProjectAlpha) or use the browse button to pick one visually.

The Preview

After scanning, the tool displays every file it found along with its proposed destination. Files are grouped by their parent folder.

Summary Chips

At the top of the preview, colored summary chips show the breakdown of classified files:

  • How many files are headed to \images
  • How many files are headed to \floorplans
  • How many files are headed to \reports
  • How many files will be skipped (already organized or excluded)

Default Classification

DestinationFile Types
\images.png, .jpg, .jpeg, .gif, .bmp, .tiff, .svg
\floorplans.dwg, .dxf, plus PDFs without report keywords in the filename
\reports.docx, .xlsx, .pptx, .csv, plus PDFs with report keywords
Tip: .esx files are never moved. They always stay in place so your Ekahau project structure is never disrupted.

Changing Destinations

Every file row has a dropdown that lets you override where it will be moved.

Per-File Dropdowns

Click the destination dropdown on any file row to choose a different subfolder. The available destinations are \images, \floorplans, and \reports, plus a Skip option to leave the file where it is.

Bulk Selection

  • Click a file row to select it.
  • Shift-click another row to select a range of files at once.
  • With files selected, use the bulk destination buttons to set all selected files to the same destination in one click.
Tip: Shift-click range selection is the fastest way to reclassify a batch of files. Select the first file, hold Shift, click the last file, then hit a bulk destination button.

Executing the Move

When you click Organize, the tool:

  1. Creates destination subfolders (\images, \floorplans, \reports) if they do not already exist.
  2. Moves each file to its assigned destination.
  3. Skips any files marked as Skip or already in the correct location.
  4. Reports a summary of how many files were moved, skipped, or encountered errors.
Note: Files are moved, not copied. The originals will no longer be in their previous location. Make sure you are happy with the preview before clicking Organize.

Settings

Open Settings to customize how files are classified.

File Type Mappings

The settings panel lets you change which file extensions map to which destination folder. You can:

  • Add new extensions to an existing destination.
  • Move an extension from one destination to another.
  • Remove extensions you do not want classified.

Destination Folders

The three default destinations use backslash notation: \images, \floorplans, and \reports. These subfolders are created relative to each site folder during the organize step.
