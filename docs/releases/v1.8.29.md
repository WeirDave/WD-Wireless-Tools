WD Wireless Tools now has one cohesive visual identity across the entire suite. This release replaces the mix of legacy artwork and emoji placeholders with a complete v8.0 branded logo family, updates every active surface and the README, and adds a theme-aware favicon.

Suite `v1.8.28 → v1.8.29`.

## Cohesive v8.0 logo family

- WD Wireless Tools — blue-and-green WD mark with “Wireless Tools” in white underneath
- Cloud Manager — branded cloud-management mark
- Quick Walls — branded floor-plan mark
- Squirrel — Wi-Fi signal moved to the squirrel’s tail
- Scale — filled ruler body with a full end-to-end measurement arrow
- Report — dedicated reporting mark replacing the clipboard emoji

Production-ready assets use the consistent `name-v8.0-size.png` naming convention. Working concept boards and transparent production copies are retained under `images/` for future brand development.

## Branding throughout the suite

- Replaced legacy artwork on the desktop home page, hosted landing page, tool cards, login/setup screens, drop zones, headers, report output, and hosted desktop-only pages.
- Replaced the Scale and Report emoji placeholders with their finished branded artwork.
- Updated the README hero and all five tool images to the new production assets.
- Updated the hosted Open Graph image to the v8.0 suite logo.

## Theme-aware favicon

- Added PNG, Apple touch, and multi-resolution ICO favicon assets.
- Added a solid-white favicon family for visibility on dark browser chrome.
- The favicon now follows the application theme immediately: white in dark mode and blue/green in light mode.
- Added favicon declarations to every active tool, guide, and hosted page.
