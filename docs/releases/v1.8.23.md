Three Duplicates polish tweaks from Dave's v1.8.22 review.

Suite `v1.8.22 → v1.8.23`. WD Cloud Manager `v3.32 → v3.33`.

## Changes

**1. Rows now read as rounded cards with breathing room.**
Was: rows sat flush against each other with a bottom border. Dave: *"I like the ones that round on the edges so that the row looks like a card."* Now:
- `.dup-items` is a flex column with `gap: 6px` and `padding: 8px`.
- Each `.dup-item` has `border-radius: 6px` and its own `background: var(--surface)`.
- No more `border-bottom` between rows — the 6 px gap does the separating.
- Bookend colored borders (8 px blue for cloud, 8 px green for local) now round at the corners because border-radius applies to the outer edge.

**2. Warmer, more orange panel color.**
Dave: *"The color looks more yellow now and less orange. I liked the other orange color better because it matched the orange that you have on the tab and the buttons better."*

`var(--amber-bg)` (`#2a2211` dark / `#fffbeb` light) was too muted — read as brown/yellow rather than orange. Introduced a dedicated token `--dup-panel`:

- Dark: `#3a2510` — warmer, tilts toward the tab/button orange.
- Light: `#fde4bd` — visibly orange-tinted on white, not just off-white.

Both `.dup-head` and `.dup-explain` use this token, so they stay pixel-identical (verified — dark `rgb(58, 37, 16)` on both; light `rgb(253, 228, 189)` on both).

**3. White workspace on the Duplicates tab.**
Dave: *"the other tabs (sites/projects) use a white workspace this has a grey workspace?"* — the app's `--bg` grey was showing between clusters.

`.dup-container` now has `background: var(--surface)`. The clusters float on a surface-colored workspace, matching what Sites/Projects show.

Verified light-mode `containerBg: rgb(255, 255, 255)` — pure white workspace.

- `web/assets/wd-tools.css`:
  - New `--dup-panel` custom property (dark/light).

  - `.dup-item` gained `border-radius: 6px` and an explicit `background: var(--surface)`. `border-bottom` between rows removed. Hover uses `filter: brightness(1.08)` instead of a bg change so the row keeps its identity.
  - `.dup-container` gained `background: var(--surface)`.
