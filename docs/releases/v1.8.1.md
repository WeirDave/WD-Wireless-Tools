Two fixes from real-world use of v2.0 on the Anduril IRVN1 site:

- The sidebar radio-button list didn't scale past two reports. Replaced with a **dropdown**. Adding a third report no longer eats vertical space.
- The BOM listed the **AP-655's four integrated antennas** (2.4/5/6/BLE) as separate line items. Integrated antennas ship with the AP — you don't buy them. **Always excluded now.**

Suite `v1.8.0 → v1.8.1`. WD Report `v2.0 → v2.1` (minor — UX + correctness).

## Dropdown picker

- `web/assets/js/report.js` — `renderReportPicker()` now emits `<select class="rep-select">` with the report's description rendered underneath. `selectReport()` re-renders the picker on change so the description tracks the selection.
- `web/assets/wd-tools.css` — added `.rep-select` (matches the existing `.template-select` treatment) and `.rep-picker-desc`. `.rep-radio-desc` kept — still used elsewhere.
- Because the picker is a `<select>`, adding a fifth or fifteenth report is trivial. The old radio list was already getting cramped at two.

## BOM: integrated antennas excluded, always

- `renderBomReport()` — added `isExternalAntenna(a)` guard: `apCoupling` must match `/EXTERNAL|DIRECT_ATTACH/i`. Everything else (INTEGRATED, INTERNAL, null, missing) is treated as internal and dropped.
- Removed the **"External antennas only"** sidebar toggle — the filter is now the only behavior. The BOM registry entry has `sidebar: []`.
- Section header renamed from "Antennas" → **"External Antennas"**, with an italic sub-note: *"Integrated antennas are omitted — they are not procured separately."*
- Empty state re-worded: *"No external antennas — all APs use integrated antennas that ship with the unit."* When the list is empty, the total row is also hidden (a "0 Total external antennas" footer read as broken).

For the Anduril IRVN1 project (26 Aruba AP-655s, all integrated antennas), the BOM now shows one line — **26 × Aruba AP-655** — and an empty external antennas section with the explanatory note. That matches what you actually buy.

## Verification

- `#reportSelect` has 2 options ("Directional Antenna Installation", "Bill of Materials"). Selecting BOM swaps the description text under the dropdown.
- `isExternalAntenna` regex tested against all realistic Ekahau values: `INTEGRATED_ANTENNAS` → false, `EXTERNAL_ANTENNAS` → true, `DIRECT_ATTACH` → true, `INTERNAL` → false, `null` → false.
