Four small Duplicates polish tweaks from Dave's v1.8.20 review.

Suite `v1.8.20 → v1.8.21`. WD Cloud Manager `v3.30 → v3.31`.

## Changes

**1. Cluster headers now use the same amber wash as the info panel.**
Was purple. Dave: *"use the same orange background you have on the info section on each of the headers instead of the purple."* Now:
- Header background: `rgba(217,119,6,0.08)` dark / `0.10` light — matches `.dup-explain` exactly.
- Hover: `rgba(217,119,6,0.14)` dark / `0.16` light.
- Title text stays default `--text` (from v1.8.20). Purple is gone from the page.

**2. Row bookends widened 4 px → 8 px.**
Cloud rows: 8 px `--blue` border-left AND border-right. Local rows: 8 px `--green` on both sides. The thicker tips read as proper bookends rather than accent lines. Base row padding compensated (`padding-left: 8px; padding-right: 8px`) so content still lines up.

**3. Info panel border-left accent removed.**
Was 4 px on the left, 1 px on the other three sides. Now uniform `1px` all around — no more asymmetric bookend on the "What am I looking at?" box.

**4. "Delete all N extras" toolbar button vertically centered too.**
Was `left: 50%; transform: translateX(-50%)` — horizontally centered but pinned to the top of the toolbar. Now `left: 50%; top: 50%; transform: translate(-50%, -50%)` so it sits dead-center vertically within the toolbar's height as well.
